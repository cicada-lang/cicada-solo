# 蝉语 / Cicada Language

## Usage

### Command Line Tools

- Install: `npm install @cicada-lang/cicada --global`
- Commands:
  ```
  cic check-library <config-file>  Check all files in a library
  cic run <file>                   Run a file
  ```
